电子书均来源于公开的互联网，仅为本人存档之用途，如有侵权，请提issue，同时也欢迎提PR增加资料

本项目使用到了超星云盘直链下载，用于大文件的存储

**相关项目：** <https://github.com/shuosc/libshu> 该项目存放了部分课程的相关资料，如习题、答案等
